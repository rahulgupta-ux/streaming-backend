# streaming-backend
Backend API for a Netflix-style video streaming app. Includes user auth (JWT), video metadata APIs, watch history, and HLS streaming support.
## Tech Stack
- Node.js + Express
- PostgreSQL / MySQL
- JWT Authentication
- FFmpeg for HLS video encoding
- Nginx for serving HLS streaming segments

## Features
- User registration & login
- Video metadata API
- Continue watching / watch history
- Admin upload endpoint
